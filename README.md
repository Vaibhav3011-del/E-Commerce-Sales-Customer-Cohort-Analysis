# 📊 E-commerce Sales & Customer Cohort Analysis

An end-to-end data analytics project that analyzes e-commerce sales data to uncover customer purchasing behavior, sales performance, and customer retention trends. The project combines Python, SQL, Microsoft Excel, and Power BI to transform raw data into meaningful business insights through data analysis and interactive dashboards.

---

## 📌 Project Overview

Understanding customer behavior is essential for improving sales performance and increasing customer retention. This project focuses on analyzing an e-commerce dataset to identify purchasing trends, evaluate customer retention using cohort analysis, and build interactive dashboards for business decision-making.

---

## 🎯 Project Objectives

- Analyze overall sales performance and revenue trends.
- Identify customer purchasing patterns.
- Perform customer cohort analysis to measure retention.
- Evaluate key business performance indicators (KPIs).
- Create interactive dashboards for business insights.
- Generate actionable recommendations based on data.

## 🔄 Project Workflow

### 1️⃣ Data Cleaning

- Removed duplicate records
- Handled missing values
- Corrected inconsistent data formats
- Converted data types
- Prepared the dataset for analysis

### 2️⃣ SQL Analysis

Performed SQL queries to analyze:

- Total Revenue
- Monthly Sales
- Top Selling Products
- Customer Purchase Frequency
- Average Order Value
- Revenue by Category
- Regional Sales Performance
- Customer Lifetime Value (CLV)

### 3️⃣ Exploratory Data Analysis (EDA)

Using Python:

- Sales Trend Analysis
- Monthly Revenue Analysis
- Product Category Performance
- Customer Segmentation
- Profit Analysis
- Regional Sales Distribution

### 4️⃣ Customer Cohort Analysis

Performed cohort analysis to evaluate customer retention over time.

Calculated:

- Customer Retention Rate
- Repeat Purchase Rate
- Cohort Revenue
- Customer Lifetime Trends

### 5️⃣ Dashboard Development

Created an interactive Power BI dashboard containing:

- Executive Sales Summary
- Revenue Trends
- Customer Analysis
- Product Performance
- Cohort Retention Heatmap
- Regional Performance
- KPI Cards
- Interactive Filters & Slicers

## 🚀 How to Run the Project

1. Clone the repository

